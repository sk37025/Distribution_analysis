## 왜 우리는 분포(distribution)를 공부해야 하는가? 

- 통계학에서는 분포를 통해서 관찰된 값(sample)이 얼마나 큰 지, 작은 지를 알 수 있음    
    > 분포를 모르면 관찰된 값이 얼마나 큰 지, 작은 지를 얘기할 수가 없음. 
- 분포는 분석가로 하여금 가설의 통계적 유의성을 논하는 것을 가능하게 함
